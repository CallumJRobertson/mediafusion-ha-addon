# Changelog

## [1.0.0] - 2024

### Added
- Initial release of MediaFusion Home Assistant Add-on
- Embedded PostgreSQL database for data storage
- Embedded Redis for caching and task queue
- WireGuard VPN support with kill switch option
- Cloudflare Tunnel compatibility
- Support for debrid providers:
  - Real-Debrid
  - AllDebrid
  - Premiumize
- Scraping sources:
  - Torrentio integration
  - Zilean DMM integration
  - MediaFusion instance scraping
- Optional Prowlarr integration
- Health monitoring and auto-restart
- Optimized for amd64 architecture (Intel MacBook Air)
- Resource-efficient configuration for limited hardware
- Persistent storage for database and cache
- Comprehensive logging with configurable levels
- Ingress support for Home Assistant UI integration
